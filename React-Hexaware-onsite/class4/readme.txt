# JSX
JSX = Javascript + XML
extension to the javascript language syntax
write xml like code for elements and components
JSX is not necessay to write in react app this is for our convinenece only
JSX can be consider as syntactical expression of javascript syntax, used to create a react element

# Why JSX?
- faster then javascript in compilation and execution
- ensure optimization during transpilation of the code to JS
- ensure better optimization as it allow react to hightlight usefull error ad message of warning at compile time
- using JSX develope find esiter to worl with the component of react having UI inside the js code
